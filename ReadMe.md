# Shortify

  <strong>A modern, fast, and secure URL shortener with analytics, QR code generation, and custom short links.</strong>
</p>

# Installation

Clone the repository.

```bash
git clone https://github.com/RealKaien/Shortify.git
```

Go to the project directory.

```bash
cd Shortify
```

Install dependencies.

```bash
npm install
```

Create a `.env` file.

```env
DATABASE_URL=

NEXTAUTH_SECRET=

NEXTAUTH_URL=http://localhost:3000

BASE_URL=http://localhost:3000
```

Run the development server.

```bash
npm run dev
```

Open:

```
http://localhost:3000
```

---

# Build

```bash
npm run build
```

Start production.

```bash
npm start
```

# Usage

1. Paste a long URL.
2. Click **Shorten**.
3. Copy the generated short link.
4. Share it anywhere.
5. Monitor analytics from the dashboard.
